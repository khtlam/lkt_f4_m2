Part 2 - Basic Arithmetic Operations

  1. Five basic arithemetic operations performed with different operators.

    • Addition (operator is '+')
      e.g., a = a + 1, where a is a variable of numeric type.
    • Substraction (operator is '-')
      e.g., a = a - 1, where a is a variable of numeric type.
    • Multiplication (operator is '*')
      e.g., a = a * 1, where a is a variable of numeric type.
    • Division (operator is '/')
      e.g., a = a / 1, where a is a variable of numeric type.
    • Remainder (operator is '%')
      e.g., a = a % 1, where a is a variable of numeric type.
